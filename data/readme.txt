This folder contains 2 batches of touch data
Each batch contains touch data for a number of participants
For each person, we collected data for two fingers(0:thumb,1index)
For each finger we collected data for two finger regions(0:center,1:tip)
The data collected includes a sequence of capacitive images and touch force data.
Each sequence consists of two finger actions. The first part is press action and the second part is push action.
